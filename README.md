圓錐曲線截痕實驗室 (Conic Sections Lab)

這是一個基於 THREE.js 的互動式 3D 模擬器，用於視覺化圓錐曲線（Conic Sections）的形成過程。使用者可以即時調整圓錐體的開口角度和切割平面的角度及位置，觀察圓錐被平面截切後所產生的圓形、橢圓、拋物線或雙曲線。

模擬器功能與操作

您可以使用左側控制面板上的滑桿和按鈕來操作 3D 模擬。

圓錐曲線的數學原理

圓錐曲線的類型完全取決於平面角度 ($\theta$) 和圓錐開口半角 ($\alpha$) 之間的關係：

圓錐曲線類型

條件 (與 $\alpha$ 比較)

幾何特性

圓形 (Circle)

$\theta = 0^\circ$

平面與圓錐軸線垂直。

橢圓 (Ellipse)

$0^\circ < \theta < 90^\circ - \alpha$

平面與圓錐的所有母線相交，只截切單一圓錐。

拋物線 (Parabola)

$\theta \approx 90^\circ - \alpha$

平面與圓錐的一條母線平行，開口無限延伸。

雙曲線 (Hyperbola)

$\theta > 90^\circ - \alpha$

平面比圓錐母線更陡峭，同時截切上下兩個圓錐。

退化曲線

平面位置 (Offset) 接近 0

平面通過圓錐頂點，產生點、線或兩條相交直線。

視覺化圖例

模擬器中的顏色幫助您區分不同的幾何物件：

藍色圓錐: 圓錐體 (母線和內表面)。

翡翠綠平面: 截平面 (切割刀)。

紅色曲線: 圓錐截痕 (產生的圓錐曲線)。
